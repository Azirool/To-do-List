# Overview
Simple To-Do List Application
 - A basic to-do list application built with JavaScript, EJS, HTML, CSS, Express.js, and Node.js.

# Features
- EJS Templates: Utilizes EJS templates for flexible and data-driven views.
- Express.js Backend: A robust Express.js backend for handling requests and data.
- Node.js Runtime: Built on Node.js for efficient server-side JavaScript execution.
- Custom Styling: Custom CSS for a polished and visually appealing design.
- Add tasks to your to-do list.
- Mark tasks as completed.
- Delete tasks.
- Clean and intuitive user interface.
